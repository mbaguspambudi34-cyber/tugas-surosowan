# Bio Link Page

Project ini adalah halaman Bio Link Page seperti Linktree yang dibuat menggunakan HTML, CSS, dan JavaScript.

## Fitur

- Foto profil / avatar
- Nama dan bio singkat
- Tombol link sosial media
- Responsive untuk desktop dan mobile
- Menggunakan HTML5 semantic tags
- Dark mode toggle menggunakan JavaScript
- Hover animation pada tombol
- Gradient background

## Teknologi

- HTML5
- CSS3
- JavaScript

## Cara Menjalankan

1. Download atau clone project ini.
2. Buka file `index.html` di browser.
3. Halaman Bio Link Page siap digunakan.

## Cara Deploy ke GitHub Pages

1. Buat repository baru di GitHub.
2. Upload file `index.html` dan `README.md`.
3. Buka menu `Settings`.
4. Pilih `Pages`.
5. Pada bagian `Branch`, pilih `main`.
6. Klik `Save`.
7. Tunggu beberapa saat sampai link GitHub Pages aktif.
